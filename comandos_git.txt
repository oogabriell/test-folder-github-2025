git init .  -> cria um repo git na pasta atual
git status  -> mostra o status do trabalho
git commit -m "msg" -> faz o commit do que estava pendente de ser comitado
git add nome_arquivo -> adiciona um arquivo para ser comitado
rm -rf -> remove de forma recursiva e forçada todos os diretórios
git log - > mostra o histórico de commit

##Fluxo de trabalho


1. criar o diretório do projeto
2. iniciar o git dentro do diretório criado (git init.)
3. edita ou cria arquivos
4. git status
5. git add <nome arquivo>
6. git satus
7. git commit -m "mensagem"
8. git status
9. volta ao passo 3
